## The System
A GraphQL API for Pokemon Data. Based in the pokemon graphql server: 
https://graphql-pokemon.now.sh

## Features
- GraphQL  API
- React Apollo Client
- RealTime GraphQL Query Making View
- Query Results in JSON Panel
